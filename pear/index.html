Kelpie PEAR channel

Registering the channel: 
pear channel-discover getkelpie.com/pear
Listing available packages: 
pear remote-list -c kelpie
Installing a package: 
pear install kelpie/package_name
Installing a specific version/stability: 
pear install kelpie/package_name-1.0.0
pear install kelpie/package_name-beta
Receiving updates via a feed: 
http://getkelpie.org/pear/feed.xml
